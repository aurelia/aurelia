# AUR0106: Invalid Assignment to `$host`

## Error Message

`AUR0106: Ast eval error: invalid assignment. "$host" is a reserved keyword.`

## Description

This error occurs when an expression attempts to assign to `$host`. `$host` is a reserved keyword and is read-only.

## Solution

- Remove the assignment and assign to a real property on your view-model instead.
