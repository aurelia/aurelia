# AUR0170: Unexpected Token in Destructuring

## Error Message

`AUR0170: Expression error: unexpected "{{token}}" at position "{{pos}}" for destructuring assignment in "{{expression}}"`

## Description

This error occurs when using destructuring assignment syntax in a binding expression and the parser encounters an unexpected token.

## Example Trigger

```html
<!-- âŒ Malformed destructuring -->
<div textcontent.bind="({ a, } = value)"></div>
```

## Solution

- Fix the destructuring syntax, or move destructuring logic into view-model code.

## Troubleshooting

- Avoid complex destructuring in templates; compute derived values in the view-model.
