# AUR0171: Unexpected Token in Optional Chain

## Error Message

`AUR0171: Expression error: unexpected {{token}} at position "{{pos}}" for optional chain in "{{expression}}"`

## Description

This error occurs when optional chaining (`?.`, `?.[ ]`, etc.) is used in an unsupported or malformed way.

## Example Trigger

```html
<!-- ❌ Malformed optional chain -->
<div textcontent.bind="user?."></div>
```

## Solution

- Use valid optional chaining syntax, or use guards (`if.bind`) to avoid null access.

```html
<!-- ✅ -->
<div textcontent.bind="user?.name"></div>
```
