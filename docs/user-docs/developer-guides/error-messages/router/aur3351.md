# AUR3351: Unexpected Viewport Deactivation

## Error Message

`AUR3351: Unexpected viewport deactivation outside of a transition context at %s`

## Description

This error indicates the router attempted to deactivate a viewport outside of an active transition context. This typically points to an internal state issue.

## Troubleshooting

- If you can reproduce this reliably, please report it with a minimal reproduction.
- Review custom router/viewport integrations for direct calls into viewport agent internals.
