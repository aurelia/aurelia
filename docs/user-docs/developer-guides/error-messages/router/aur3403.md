# AUR3403: Invalid Instruction Type for `toUrlComponent`

## Error Message

`AUR3403: Invalid instruction type %s for "toUrlComponent" operation. If you are seeing this error, then it is probably because of an internal bug. Please report it.`

## Description

This error occurs when `toUrlComponent` is called with an instruction type that it does not support.

## Troubleshooting

- If you call `toUrlComponent` directly, ensure you pass a supported instruction type.
- If you do not call it directly, this likely indicates a router bug; please report it with a reproduction.
