# AUR3554: Invalid Route Config Property Type

## Error Message

`AUR3554: Invalid route config property: "%s". Expected %s, but got %s.`

## Description

This error occurs when a route configuration property has an unexpected type.

## Solutions

- Fix the property type to match what the router expects.

## Troubleshooting

- Inspect the property named in the first `%s` and check the expected vs actual types.
