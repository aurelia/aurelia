# AUR3556: Unknown Route Config Property

## Error Message

`AUR3556: Unknown route config property: "%s.%s". Please specify known properties only.`

## Description

This error occurs when a route configuration object includes an unknown property.

## Solutions

- Remove/rename the unknown property so only supported properties are used.

## Troubleshooting

- Check the two `%s` values to see where the unknown property appears.
